# Stats: Insights and Overview

Visualize viewing habits and library distribution with charts and metrics.

## Status: `Released`

## Description

Users can see their library distribution by genre, their watch activity over the current year, and key metrics like total watch time and average rating.

## Key Features

- **Key Metrics**: Total watched, watchlist count, average rating, and total watch time.
- **Genre Distribution**: Horizontal bar chart showing items per genre.
- **Monthly Activity**: Bar chart showing items watched per month in the current year.
- **Top Rated**: List of the top 10 highest-rated items.
